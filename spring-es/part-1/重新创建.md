- 重新给当前索引创建别名(如果有就略过)
```shell script
Post请求 localhost:9200/_aliases
{
  "actions": [
    {
      "add": {
        "index": "twitter",
        "alias": "twitter_latest"
      }
    }
  ]
}
```

- 创建新的分词索引

```shell script
Put请求 http://localhost:9200/twitter_v2

{   
	"settings":{
		 "analysis" : {
            "analyzer" : {
                "ik" : {
                    "tokenizer" : "ik_max_word"
                }
            }
        }
	},
    "mappings":{
        "properties": {
            "user": {
                "type": "keyword"},
            "tag" : { "type" : "keyword" },
            "age" : { "type" : "byte" },
            "post_date" : { "type" : "date" },
            "message" : { "type" : "text" }
        }
    }
}
```

- 导入数据 reIndex
[导入数据](https://www.elastic.co/guide/en/elasticsearch/reference/current/docs-reindex.html)
[参考文档](https://blog.csdn.net/u013613428/article/details/78227277)
```shell script
Post请求 http://localhost:9200/_reindex

{
  "source": {
    "index": "twitter"
  },
  "dest": {
    "index": "twitter_v2"
  }
}
```

- 改变索引别名
```shell script
Post  http://localhost:9200/_aliases
{
    "actions": [
        { "remove": {
            "alias": "twitter_latest",
            "index": "twitter"
        }},
        { "add": {
            "alias": "twitter_latest",
            "index": "twitter_v2"
        }}
    ]
}
```
